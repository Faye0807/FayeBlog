vue input file 多次上传不触发change事件问题

有一下几种方法（参考链接）：

  • 清空input的value值
  • 替换掉原来的input 渲染一个新的input
  • 将input放到一个表单里，change事件触发时将表单reset
  • 替换原input，渲染新input并重新绑定change事件（这个方法对我有效）
