css居中

css居中的方法有多种，需要居中的情形也有多种，下面就来总结一下

inline-block 的居中

  • 多个inline-block 的水平居中text-align: center
  • 多个inline-block 的垂直居中vertical-align: middle

块元素的居中

无论父元素、子元素的宽度是否都已知，都可以通过margin: auto；来水平居中
注意margin、padding的百分比是基于其包含块宽度的百分比

父元素、子元素定宽定高

  • 通过设置margin值，计算出垂直、水平居中的margin值达到居中的效果

父元素大小未知、子元素大小确定

知识点： absolute定位的top、right等值，百分比时，取自其相对元素的宽度、高度的百分比；

  • 父元素相对定位；子元素绝对定位 + margin负值

父元素、子元素大小未知

  • 父元素相对定位；子元素绝对定位(子元素left、right、bottom、top均为0) + margin: auto
  • 父元素相对定位；子元素绝对定位 + 位移

知识点： 定位百分比的值取自“其父元素”，而位移变换的半分比，取自作用当前元素的百分比

  • 父元素flex

其它居中方法

  • 父元素table-cell+vertical-align;
  • 添加一个伪元素，作为对照物+vertical-align来垂直居中;
